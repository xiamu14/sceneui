# Button

Buttons allow users to perform actions and choose with a single tap.

```js
import Button from '@sceneui/button';
```

### Default

<Button>确认</Button>

```js
<Button>确认</Button>
```

### Rounded

<Button round>确认</Button>

```js
<Button round>确认</Button>
```

### Icon

<Button icon={<Check />}>完成</Button>

```js
import { Check } from 'react-feather';

<Button icon={<Check />}>完成</Button>;
```

Use **className** property to adjust the style of Button more flexibly.
