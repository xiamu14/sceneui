# Button

Buttons allow users to perform actions and choose with a single tap.

```js
import Button from '@sceneui/button';
```

### Default

<Button>确认</Button>

```js
<Button>确认</Button>
```

### Round

<Button round>确认</Button>

```js
<Button round>确认</Button>
```

### Solid

<Button solid>确认</Button>

```js
<Button solid>确认</Button>
```

With Round

<Button solid round>
  确认
</Button>
```js
<Button solid round>
  确认
</Button>
```

### Custom style

Use **style** property to adjust the style of Button more flexibly.

<Button style={{ width: 96, height: 36 }} disable>
  确认
</Button>

```js
<Button style={{ width: 96, height: 36 }} disable>
  确认
</Button>
```

With Icon

<Button style={{ width: 36, height: 36 }}>
  <ArrowLeft />
</Button>

```js
<Button style={{ width: 36, height: 36 }}>
  <ArrowLeft />
</Button>
```

## Props

```js
export interface Props {
  round?: boolean;
  solid?: boolean;
  className?: string;
  disable?: boolean;
  style?: CSSProperties;
}
```
